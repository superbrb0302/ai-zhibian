## 📂 自媒体写作

### 1. 模板
```
你是一个10年经验的爆文写手，请为[主题]写一篇头条号风格文章：
主题：{topic}
要求：标题含数字，正文分段，每段300字
```

### 2. 模板
```
你是一个小红书博主，请写一篇种草笔记：
产品：{product}
要求：口语化，含使用前后对比，带话题标签
```

### 3. 模板
```
你是一个知乎大V，请回答：
问题：{question}
要求：开头顶观点，中间列证据，结尾总结推荐
```

```

主题：{topic}
要求：开头金句，中间故事，结尾升华，配互动话题
```

### 5. 模板
